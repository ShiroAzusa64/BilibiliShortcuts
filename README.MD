# B站快捷键插件

适用于天选系列键盘，专门设计为抱着电脑躺床上刷b站，默认键位适用于b站两排三列视频快速触发。程序未写死视频映射，可在json配置文件添加数组自动按照视频序号倍增或减少视频触发快捷键

## 默认键位支持:

主界面支持视频推送刷新和回退推送历史(数字1，2键，箭头键，10条历史记录)，按照 .;[ /'] 对应第一排第123个视频类推，左右都有适配

视频界面支持按"Shift 1 2"三倍速播放,"z /" 点赞 "\ ESC" 关闭视频

动态界面支持箭头和"1 2"键自动点头像查看动态

## 配置说明:

在popup页面支持以json格式更改快捷键配置，请参照原始schema,若需要使用默认值请清空文本框并点击保存。
